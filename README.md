As part of my ECON 4200 (Senior Seminar) course, I built a metro-month panel linking Zillow home values to every U.S. stadium and arena opening since 2000, and estimated causal effects using staggered Difference-in-Differences and event study models. I found that home value gains from venue openings are small and fade over time, offering realistic evidence for voters and planners about the economic impact of venue construction.
